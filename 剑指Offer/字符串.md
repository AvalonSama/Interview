#### 题4 替换空格

**题目：**把字符串中的空格替换成'20%'。

<details>

<summary><b>解答</b></summary>

**方法一：**
```python
s = input()
return s.replace(' ', '%20')
```

**方法二：**
```python
s = input()
res = ''
for i in range(len(s) - 1, -1, -1):
    if s[i] == ' ':
        res = '20%' + res
    else:
        res = s[i] + res
print(res)
```
