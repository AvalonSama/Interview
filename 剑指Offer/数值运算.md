#### 题13 二进制中1的个数

**题目：**输入一个整数，输出该数二进制表示中`1`的个数。

<details>

<summary><b>解答</b></summary>

**方法一：**左移方法，通过不断将`1`左移与`n`做与运算判断当前位置是否为`1`。

```python
def number_of_1(n):
    count, flag = 0, 1
    if n < 0:
        n = n & 0xFFFFFFFF
    for i in range(32): # 32位
        if n & flag:
            count += 1
        flag = flag << 1
    return count
```

**方法二：**利用与运算，`(n - 1) & n`将最右侧为`1`的位置的值，改为`0`。

```python
def numberof1(n):
    count = 0
    if n < 0:
        n = n & 0xFFFFFFFF
    while n:
        count += 1
        n = (n - 1) & n
    return count
```
